Project Overview
This demo shows a plethora of engaging mechanics, predominantly emphasizing immersive third-person gameplay elements. It was made entirely in UE5. Some 3D assets are from Sketchfab or Quixel and the main character, as well as his animations are from the ALS V4.
The demo map "Echoes of Metropolis" is inspired by the video game "The Last of Us".
"Echoes Of Metropolis" - Level Design by Windokk
The Demo
June 2023
This demo features various third-person gameplay mechanics. They are grouped in 7 categories : Locomotion System, Interaction System, Health System, AI System, RPG System, Items System and Vehicles System.
Locomotion System :
It features basic and advanced movement animations and actions :
- Forward/Backward, Left/Right Movement
- Jumping/Vaulting/Mantling
- Crouching and Taking Cover
- Foot IK
- Swimming System
Interaction System :
For now, the interaction system has only one interaction : The lever
When you press the interaction key, you push or pull a lever, that can activate other objects, such as a street lamp.
Health System :
This system is only a "damage" system for now but i plan to add "health restoring" items in the future.
Different ways of taking damage :
- Fall Damage (relative to the fall height)
- Fire Damage (fire can be stopped by going into the water
AI System :
Nothing to show right now, still working on it...
RPG System :
Has been removed but it added an XP system and Leveling System with UI and Sound effects.
Items System :
The item system adds two types of items : weapons and tools.
In the weapon type : there are firearms and melee weapons.
By pressing the interaction key, the player can pick them up, then press "R" to equip them for his inventory.
He can then use the weapons on "dummies". Some weapons like the knife have custom properties such as stealth assassinations. There are 4 melee weapons : Fists, Knife, Axe, Baseball Bat.
There are 3 firearms and they all work differently : The assault riffle (M4A1) shoots rapidly but makes less damage, the shotgun is imprecise (lots of small bullets parts) but makes a lot of damage, and the pistol shoots slowly but is precise and makes a lot of damage.
All of these weapons can break the glass placed on the map when attacking.
There are also 2 tools on this demo : the flashlight and the binoculars. They act as in real life.
Vehicle System :
There are 3 motor vehicles that can be driven the same way (accelerate, brake, turn...). They can also be damaged and be unusable when fire is coming out of the engine.
A horse has recently been added to the demo but it will be enhanced in the future.
